---
description: Start or resume a software architecture patterns tutoring session with the arch-tutor agent
argument-hint: "[starting-topic]"
---

Invoke the `arch-tutor` agent to run an architecture patterns tutoring session.

The arch-tutor agent will use the arch-teach skill to manage the session — resuming from `lastsession.md` if one exists, or running the kickoff sequence to build a roadmap and project.

Pass any arguments as the starting topic or focus area (e.g., "hexagonal", "event-driven", "microservices boundaries").
