# my-plugins

![Build](https://github.com/rgoshen/my_plugins/actions/workflows/validate.yml/badge.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Rick Goshen's personal Claude Code plugin marketplace.

## Install this marketplace

```
/plugin marketplace add rgoshen/my_plugins
```

## Available plugins

| Plugin | Version | Description |
|---|---|---|
| [cs-tutor](./plugins/cs-tutor/README.md) | ![cs-tutor version](https://img.shields.io/badge/cs--tutor-v0.0.2-blue.svg) | Senior-engineer mentors for architecture patterns and programming language learning |

## Install a plugin

```
/plugin install cs-tutor@my-plugins
```
